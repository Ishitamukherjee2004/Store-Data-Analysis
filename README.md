This project analyzes a retail store dataset to uncover sales trends, profit patterns, and customer behavior. 
The goal is to generate insights that can help improve business decisions.


Dataset Information
- Dataset: Superstore Sales Dataset
- Source: Kaggle
- Rows: ~4500
- Columns: Order ID, Order Date, Ship Date, Ship Mode, Customer ID, Customer Name, Segment, Country, City, State, Postal Code, Region, Product ID, Category, Sub-Category, Quantity, Discount, Profit


Objectives / Questions
Key questions answered:
- What is the variation between Ship Mode and Category?
- Which category performs best?
- Which city generates the highest revenue?
- Which Category is more profitable?
- Which customer segment is most profitable?


Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook (Anaconda)
